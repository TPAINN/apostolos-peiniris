Personal portfolio and CV site — hand-built, single `index.html`, with custom cursor effects, smooth scroll animations, and downloadable certificates (secondary school diploma, language certifications, military service, Java certification).

## Features

- Zero-dependency, zero-build static site
- Custom cursor interactions and scroll-based animations
- Downloadable certificate PDFs (education, languages, military, technical)

## Stack

| Layer | Tech |
|---|---|
| Markup/Styling/Logic | Vanilla HTML, CSS, JS (single file) |
| Hosting | Vercel |

## Run locally

Just open `index.html` in a browser — no build step required.
